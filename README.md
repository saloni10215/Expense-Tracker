# 💰 Expense Tracker

A simple and responsive Expense Tracker web application built using HTML, CSS, and JavaScript.

Track your daily expenses, organize them by category, search and filter records, and keep your data saved using Local Storage.

---

## 🚀 Features

### Expense Management
- Add new expenses
- Edit existing expenses
- Delete expenses
- Automatic total expense calculation

### Search & Filter
- Search expenses by name
- Filter expenses by category
- Real-time updates

### Data Persistence
- Save expenses using Local Storage
- Data remains after page refresh

### User Experience
- Responsive design
- Modern UI
- Enter key support for quick expense entry
- Clean and easy-to-use interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
